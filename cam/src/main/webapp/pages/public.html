Asset Registry

Asset Registry is a web application which provides a user interface for exploiting the Asset Registry API interactively. Asset Registry allows user to create, read, update and delete Classes, Models, Assets, Attributes, Relationships and Domains through a web interface.

Asset Registry API

Asset Registry API is a web application with no user interface. It provides a REST-based abstraction on top of the reference ontology that is stored in the RDF4J repository (not included: see installation instructions below). By means of API calls, applications can query the reference ontology: CRUD operations are allowed on Classes, Models, Assets, Attributes, Relationships and Domains. Asset Registry API also implements the integration with the FIWARE Orion Context Broker and with FIWARE security.
